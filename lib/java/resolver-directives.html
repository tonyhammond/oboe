Oboe/Ruby - Service Directives

The following service directives can be attached to the OpenURL to modify the resolution behaviour:

  srv_id=_ri:oboe:to_key   Render OpenURL in Key encoding
  srv_id=_ri:oboe:to_xml   Render OpenURL in XML encoding
  srv_id=_ri:oboe:to_rdf   Render OpenURL in RDF encoding
   
  srv_id=_ri:oboe:no_val   Don't validate
  srv_id=_ri:oboe:no_ref   Don't resolve references to external resources
