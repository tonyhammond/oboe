Oboe/Java - Simple Resolver

Below is a complete example of a simple resolver in Java. The toXml() method returns an OpenURL object as an XML doument. 
  import 'javax.servlet.*'
  import 'oboe.lib.*';

  OpenURL o = new OpenURL();

  System.out.print "content-type: application/xml\n\n";
  System.out.print o.new(args).toXml();
