Oboe/Ruby - Example

Below is a complete example of a simple parser in Ruby. The to_xml method returns an OpenURL object as an XML doument. 
#!/usr/local/bin/ruby

  require 'cgi'
  require 'oboe'; include Oboe

  cgi = CGI.new; args = cgi.params

  print "content-type: application/xml\n\n"
  print OpenURL.new(args).to_xml
