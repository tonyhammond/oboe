Oboe/Perl - Simple Resolver

Below is a complete example of a simple resolver in Perl. The to_xml method returns an OpenURL object as an XML document. 
#!/usr/local/bin/perl

  use CGI;
  use Oboe;

  $cgi = CGI.new; @nams = $cgi->parse;

  for (@nams) {
    @vals = $cgi->param($_);
    $args{$_} = [ @vals ];
  }
  print "content-type: application/xml\n\n";
  print Oboe::OpenURL->new(\%args)->to_xml;
